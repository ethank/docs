# UA Labs Organizational Design

## The Core Question

How do you staff and organize an innovation lab that:
1. Can ship 8-week experiments at speed (per your strategic framework)
2. Leans heavily on AI-Augmented Development (AIAD)
3. Remains "black-boxed" from legacy processes that would slow it down
4. Still maintains interface/support for legacy systems when needed

## Core Philosophy

**"Small teams, big AI, zero process"**

The Labs division operates on fundamentally different principles than traditional engineering:

| Dimension | Legacy Division | UA Labs |
|-----------|-----------------|---------|
| Team Size | 10-30 per team | 2-3 per pod |
| Shipping Cadence | Quarterly releases | Weekly/daily |
| Planning | Roadmaps, PRDs, reviews | 8-week experiments, fill or kill |
| Code Reviews | Multi-stage approval | AI-assisted, trust-based |
| Testing | Full QA cycles | Automated + user feedback |
| Tools | Established stack | Whatever ships fastest |
| AI Usage | Augmentation | Core methodology (AIAD) |

### Builder (2 per pod)

AIAD-native full-stack engineers who:
- Ship features daily using AI assistance
- Own entire vertical from idea to production
- Make technical decisions without committee
- Build in public (daily updates, demos)

### Amplifier (1 per pod)

Community builder + QA + content creator who:
- Manages Discord community
- Creates daily content (videos, posts, updates)
- First-line user testing and feedback collection
- Coordinates with Builders on priorities based on user signal

## Key Design Decisions

<CardGroup cols={2}>
  <Card title="Hybrid Spinout" icon="rocket">
    Successful experiments graduate to their own product teams. Labs stays lean as an incubator.
  </Card>
  <Card title="Mostly External Hiring" icon="users">
    AIAD fluency is rare in traditional orgs. Hire fresh talent with AI-native workflows.
  </Card>
  <Card title="Remote-First" icon="globe">
    Access global AIAD talent pool. Discord becomes the office. Async by default.
  </Card>
  <Card title="CTO Direct" icon="shield">
    CTO leads Labs for first 6-12 months to protect from legacy process creep.
  </Card>
</CardGroup>
